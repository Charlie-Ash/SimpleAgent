### Conclusion
The ReAct agent is goes through pure-text reasoning, which works with any LLM. However, this causes problems such as the in *Run 1* happen. Formatting strings, string parsing, and hallucination happens. 

Structured agents are better prefered. Something that can output a structured response, e.g. JSON. 

#### Pipeline working:
ReAct makes the LLM run un this loop:
```
Thought → Action → Observation → Thought → Action → ... → Final Answer
```
The LLM generates plain text like:
```
Thought: I need more information
Action: Search
Action Input: Kenya
```
Then LangChain:
1. parses the text
2. extracts:
    `Action`
    `Action Input`
3. calls your Python function
4. feeds the result back as
```
Observation: <tool result>
```
And the loop continues...
